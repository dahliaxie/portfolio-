Loops A loop makes sure that the code is repeating for however long the while statement is true. A plane will continue to fly until they reach their destination
Conditionals A conditional is an "if... then" statement, the output of the conditional is based on whether the if statement is true or not. If it's sunny I will go outside but if it's raining then I'll stay inside
Functions A function is a set of commands that are able to be reused
